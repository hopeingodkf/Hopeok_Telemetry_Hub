# Hopeok Telemetry Hub: Промислова система збору та візуалізації даних

**Hopeok Telemetry Hub** (інтеграція за протоколом DataLink) — це професійний десктопний комплекс на Python (PyQt6 / CustomTkinter), розроблений Hopeok Code Studio для інженерів, embedded-розробників та спеціалістів з IoT.

Додаток призначений для високошвидкісного моніторингу, калібрування та логування технологічних параметрів у реальному часі без навантаження на графічний інтерфейс.

---

## 🛠️ Що вміє цей інструмент?

* **Швидкісний UART потік (921600 baud):** Фонова обробка пакетів даних без затримок інтерфейсу.
* **8 незалежних DC каналів:** Моніторинг напруги, струму та потужності (INA226).
* **Промисловий тахометр (RPM):** Розрахунок обертів двигуна з урахуванням кількості полюсів.
* **Модуль ваги (Kg):** Калібрування тензодатчиків (HX711) та швидке встановлення тари.
* **Контроль AC мережі (220V):** Інтеграція з модулями PZEM для аналізу змінного струму.
* **RAW UART Діагностика:** Виведення сирих HEX-байтів у реальному часі з частотою 10 Гц.
* **Розумний експорт:** Автоматичний запис сесій у CSV-файли, повністю сумісні з Excel.
* **Кастомний стиль інтерфейсу:** Перемикання тем прямо «на льоту» (Hopeok Brand, Cyberpunk, VSCode Dark).

## 🛡️ Демонстрація системи захисту (Early Access)

Цей софт є наочною демонстрацією комерційної екосистеми безпеки розробок Hopeok Code Studio. Програма скомпільована в захищений бінарний файл за допомогою комерційного інструменту **Nuitka**, що повністю унеможливлює реверс-інжиніринг.

При запуску система виконує:

1. **Апаратну прив'язку (HWID/UID):** Жорстка прив'язка ліцензії до заліза вашого ПК та мікроконтролера.
2. **Клієнт-Серверний захист:** Валідація ліцензійного ключа через наш захищений Web-сервер (PHP/MySQL).
3. **Активну кібербезпеку:** Автоматичний бан зловмисників за IP-адресою при спробах підбору ключів чи зламу.
